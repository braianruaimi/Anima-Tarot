# Anima Tarot

Plantilla web estatica en espanol para GitHub Pages con estilo tarot profesional.

## Incluye

- Hero con CTA a WhatsApp
- Seccion sobre el negocio
- Tarjetas editables de productos o servicios
- Formulario de reserva o contacto con mensaje de exito
- Testimonios
- CTA final
- Boton flotante de WhatsApp
- Boton flotante de asistente virtual
- Chatbot simple en JavaScript
- PWA instalable con manifest y service worker

## Estructura

- css/
- js/
- images/
- index.html
- manifest.webmanifest
- service-worker.js

## Personalizacion rapida

- Colores base en css/styles.css dentro de :root
- Textos y secciones en index.html
- Respuestas del chatbot en js/chatbot.js
- Numero de WhatsApp en los enlaces de index.html

## Publicacion en GitHub Pages

1. Sube estos archivos al repositorio.
2. Activa GitHub Pages desde Settings > Pages.
3. Usa la rama main y la carpeta raiz.

## Nota

Para una PWA instalable mas robusta conviene servirla con HTTPS, que GitHub Pages ya proporciona.
